About the Project

Crop Recommendation
using Federated Learning

A privacy-preserving deep learning system that trains across distributed farm clients to recommend the best crops without centralizing sensitive agricultural data.

Technology Stack

Core Technologies

🧠

Multilayer Perceptron (MLP)

Deep learning model with 64→32→22 neuron architecture trained using Adam optimizer with categorical cross-entropy loss for multi-class crop classification.

🔒

Federated Learning (Flower)

Privacy-preserving distributed training using the Flower (flwr) framework with FedAvg aggregation across 3 independent client nodes.

🌐

Flask Web Application

Lightweight Python web framework providing REST API and user interface for real-time crop prediction, registration and login.

🗄️

SQLite Database

Lightweight embedded database storing user registrations permanently. No server required — data stored in a local users.db file.

🐍

TensorFlow / Keras

Backend deep learning framework for building, training and saving the MLP model with Adam optimizer and cross-entropy loss.

📈

Scikit-Learn + Matplotlib

Used for data preprocessing (StandardScaler, LabelEncoder) and generating performance metrics and visualization plots.
